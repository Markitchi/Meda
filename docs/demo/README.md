The demo video provides a comprehensive walkthrough of Meda's key features:

#### 2. Dashboard Overview (0:30 - 1:00)
- Main dashboard interface
- Statistics and metrics
- Recent activity

#### 3. Patient Management (1:00 - 2:00)
- Creating new patient records
- Viewing patient details

#### 4. Medical Image Upload & Analysis (2:00 - 3:00)
- Drag-and-drop image upload
- Image viewer with zoom/pan

#### 5. AI-Powered Diagnostics (3:00 - 4:00)
- Comprehensive diagnosis generation
- Differential diagnoses
- Priority-based recommendations
- Suggested tests with justifications

#### 6. Collaboration Features (4:00 - 4:30)

- PDF report creation
